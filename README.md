# essential-SCSS

> essential-SCSS

## 安装| Build Setup

``` bash
npm install essential-SCSS
```
## 项目引入 | Import

在项目中引入flexible(如果使用flexible作为移动端适配方案),
引入style.scss的默认样式（包含normalize.css的重置样式）
```ecmascript 6
import 'essential-SCSS/flexible'
import 'essential-SCSS/style.scss'
```
引入项目提供的scss方法和类
```scss
@import "essential-SCSS/import.scss";
```
